🚀 Como Integrar ao Projeto

Passo 1: Adicione os Scripts

Adicione ao final do seu index.html (ou arquivo base):

HTML
<!-- Chatbot BMO -->
<link rel="stylesheet" href="chatbot.css">
<script src="chatbot-service.js"></script>
<script src="chatbot.js"></script>

Passo 2: Copie os Arquivos

  1. chatbot-service.js - Serviço de integração com API
  2. chatbot.js - Lógica do widget
  3. chatbot.css - Estilos e animações

Passo 3: Configure o Perfil do Usuário

O chatbot carrega automaticamente do localStorage. Certifique-se de que seu auth.js salva os dados do usuário assim:

JavaScript
// Após login bem-sucedido
const userData = {
  nome: "João Silva",
  email: "joao@example.com",
  tipo: "freelancer", // ou "empresa"
  habilidades: ["JavaScript", "React", "Node.js"]
};
localStorage.setItem('user', JSON.stringify(userData));

Passo 4: Teste o Widget

  1. Abra o seu site em navegador
  2. Você verá um botão roxo flutuante no canto inferior direito
  3. Clique para abrir o chat
  4. Digite uma mensagem para testar

🔧 Variáveis CSS Personalizáveis

Edite em chatbot.css para customizar cores:

CSS
:root {
  --bmo-primary: #4F46E5;        /* Cor principal */
  --bmo-secondary: #10B981;      /* Cor secundária */
  --bmo-accent: #F59E0B;         /* Cor destaque */
  --bmo-white: #FFFFFF;
  --bmo-dark-gray: #6B7280;
}

🐛 Solução de Problemas

  • Widget não aparece: Verifique se os scripts estão sendo carregados (Console > Network)
  • Mensagens não são enviadas: Verifique a chave da Gemini API em chatbot-service.js
  • Estilos não aplicam: Certifique-se de que chatbot.css está carregado ANTES de outros estilos
  • Perfil não aparece: Verifique localStorage com: console.log(localStorage.getItem('user'))
